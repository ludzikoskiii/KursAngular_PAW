📝 Lekcja 6 – Template Form & Reactive Form

1. Template Form

Dane formularza wiążemy z obiektem przez [(ngModel)]

@if (imieRef.invalid && imieRef.touched) {
Imię jest wymagane (min. 3 znaki)
}
@if (emailRef.invalid && emailRef.touched) {
Podaj poprawny adres email
}

2. Reactive Form

Formularz definiujemy w TypeScript przez FormGroup i FormControl

@if (imie?.invalid && imie?.touched) { @if (imie?.errors?.['required']) {
Imię jest wymagane
} @if (imie?.errors?.['minlength']) {
Minimum 3 znaki
} }
@if (email?.invalid && email?.touched) {
Podaj poprawny email
}
@if (haslo?.invalid && haslo?.touched) {
Hasło min. 6 znaków
