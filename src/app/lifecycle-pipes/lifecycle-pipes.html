2. Pipes (rury transformujące dane)

Pipe | transformuje wartość w szablonie bez zmiany oryginału

PipePrzykładWynik
uppercase {{ "'" }}{{ tekst }}{{ "'" }} | uppercase {{ tekst | uppercase }}
lowercase | lowercase {{ tekst | lowercase }}
currency {{ cena }} | currency:'PLN' {{ cena | currency:'PLN' }}
date data | date:'dd.MM.yyyy HH:mm' {{ data | date:'dd.MM.yyyy HH:mm' }}
number {{ liczba }} | number:'1.2-3' {{ liczba | number:'1.2-3' }}
percent {{ procent }} | percent {{ procent | percent }}
slice dlugaTablica | slice:0:3 {{ dlugaTablica | slice:0:3 }}
json obiekt | json {{ obiekt | json }}
