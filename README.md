# 🅰 Angular 21 – Kurs LEARNING PARTNER (wersja kodowa)

Projekt GitHub zawierający wszystkie lekcje kursu Angular 21 jako działający kod z komentarzami.

## 📚 Spis lekcji

| Lekcja | Trasa | Temat |
|--------|-------|-------|
| Part 1 | `/zmienne` | Typy zmiennych w TypeScript |
| Part 2 | `/databinding` | Data Binding (interpolacja, property, event, two-way) |
| Part 3 | `/signal` | Sygnały i Routing |
| Part 4 | `/control-flow` | Control Flow (@if, @else, @for) |
| Part 5 | `/ng-class-style` | ngClass & ngStyle (dynamiczne CSS) |
| Part 6 | `/forms` | Template Form & Reactive Form + walidacja |
| Part 7 | `/api-get` | GET API – pobieranie danych |
| Part 8 | `/api-post` | POST API – wysyłanie danych |
| Part 9 | `/api-put-delete` | PUT & DELETE API |
| Part 10 | `/crud` | Pełne CRUD z Reactive Form |
| Part 11 | `/serwis` | Serwisy i Dependency Injection |
| Part 12 | `/signal-form` | Signal Form + walidacja |
| Part 13-14 | `/login` | Login Page + API + Auth Guard |
| Part 16 | `/lifecycle-pipes` | Lifecycle Hooks & Pipes |

## 🚀 Uruchomienie

```bash
npm install
ng serve
```

Otwórz: `http://localhost:4200`
